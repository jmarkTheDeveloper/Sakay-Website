# 🚍 Sakay Web App - Philippine Transit Directions

A web-based **Philippine Transit Direction System** that helps users find efficient routes between locations using public transportation such as MRT, LRT, and other transit options.

This project aims to replicate and simplify real-world commuting in the Philippines by providing route suggestions, estimated travel time, and distance calculations through an interactive map interface.

---

## 🚀 Features

- 📍 Input origin and destination locations  
- 🗺️ Interactive map-based navigation  
- 🚆 Support for MRT/LRT routes and transit directions  
- ⏱️ Estimated travel time and distance  
- 🔄 Route suggestions and alternatives  
- 💻 Modern and responsive user interface  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript *(or React / Next.js if applicable)*  
- **Backend:** Node.js / Express *(if applicable)*  
- **APIs:** Map and routing services  
- **Version Control:** Git & GitHub  

---

## 🎯 Purpose

This project was developed to:

- Learn full-stack web development  
- Practice API integration  
- Build a real-world useful application for commuters in the Philippines
